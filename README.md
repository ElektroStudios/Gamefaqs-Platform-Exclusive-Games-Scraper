## 👋 Introduction

**GameFAQs Platform Exclusive Games Scraper** is a single-purpose console utility developed in VB.NET. It automatically crawls the video game lists of GameFAQs for any given platform, filters the titles based on their exclusivity status, and formats the collected data directly into clean, structured Markdown tables ready for GitHub documentation.

## 💡 Motivation

Compiling accurate and exhaustive lists of platform-exclusive video games is a tedious and slow process. Doing it manually requires hunting down thousands of titles across multiple web pages, verifying regional releases, filtering out multi-platform ports, and fixing text alignments inside giant Markdown files. 

This tool was built to automate that heavy lifting and eliminate the manual copy-paste overhead. Because crawling thousands of game entries from a comprehensive database requires a systematic, step-by-step approach to safely parse HTML structures and preserve data integrity, this utility handles the long-running extraction process in the background, providing a reliable backbone to build and maintain complete platform exclusivity repositories.

##### ⚡ The Real Question
###### Why take the time to map out parsing routines for legacy gaming catalogs? Because anyone who has ever tried to audit a complete console library by hand knows the mental toll of filtering regional variants and regional duplicates. This tool exists to replace human exhaustion with a cold, systematic parsing routine.

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/Gamefaqs-Platform-Exclusive-Games-Scraper/releases/latest)**.

3. Run the executable file to launch the application.

## 💪 Contributing

Your contribution is highly appreciated!. If you have any ideas, suggestions, or encounter issues, feel free to open an issue. Your input helps make this Work better for everyone. Thank you for your support! 🚀

Requesting the addition of new tables for other platforms is allowed, but such platform must exist on the Gamefaqs website.

The game entries of the tables are limited by the game entries that exist on the Gamefaqs website. This means that if you notice that an exclusive or multi-platform game is missing for a platform, it is possible that the game is cancelled, has not yet been released, or there is no entry on the Gamefaqs website. Please check if that is the case before creating an issue.
